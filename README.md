# E-Commerce Shop - FullStack PWA App with Standard Auth & Jwt Token and WebAuthn

Api -> [Click](https://github.com/jarekm04/nestjs-shop-api/tree/master) to GitHub Repo. 

# Backend
  - NestJS

# DB
  - Mongo
  - Mongo express
  - Docker

# Frontend
  - Full offline PWA with vite-plugin-pwa
  - Dexie.js - saves cart state in IndexedDB
  - React + TypeScript
  - React Context & useReducer
  - WebAuthn Register & Login - enables the use of physical, cryptographically-secure hardware "authenticators" to provide stronger replacements to passwords or 2FA
  - CSS Modules
  - Vite
    
# Testing  
  - React Testing Library
  - Vitest
  - Cypress

## Env example
```
VITE_API_HOST='http://localhost:3000/api'
```

### Run project
```
npm run dev
```

### Run unit tests
```
npm run test
```

### Run cypress tests
```
npm run cypress
```

### Build project
```
npm run build
```
